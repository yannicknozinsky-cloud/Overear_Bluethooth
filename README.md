Bluetooth-Over-Ear-Kopfhörer
 3D-gedruckte, wiederaufladbare Bluetooth-Over-Ear-Kopfhörer, die über mehrere Sensoren gesteuert werden. Eine selbst entwickelte Platine mit Shiftregistern und RGB-LEDs ermöglicht neben einem integrierten LCD Bildschirm eine visuelle Darstellung von Bass und Audio-Flow und Einstellungen.
